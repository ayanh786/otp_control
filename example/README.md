## Example App

A complete working example is included in the `/example` folder.

It demonstrates all features of the OTP Control package:

- Basic OTP input (default 6 digits)
- Custom OTP length (4 digits)
- Obscured OTP input (secure mode)
- Live OTP tracking using onChanged
- OTP completion detection using onCompleted

To run the example:

```bash
cd example
flutter pub get
flutter run
```
